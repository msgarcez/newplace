TAG Header

O novo elemento <header>do HTML5 e usado para definir o cabeçalho de uma página ou seção

Tag Section

O novo elemento <section>do HTML5 é o menos especifico entre as novas tags. A diferença do <section> um <div> e que o primeiro serve para dividir o conteudo diferentes seções , enquato o segundo dividi qualque conteudo sem uma finalidade especifica

Tag Footer

O novo elemento <footer>do HTML5 represeta o rodape de um documento ou de uma secao especifica do mesmo,

NAV

O novo elemento <nav> do HTML5 serve para agrupor uma lista de links para outraspartes do site, seja esta lista de navegação local ou global.
